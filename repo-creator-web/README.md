# Instructions to run the Flask web app for GitHub repo creation

## 1. Install dependencies

You need Python 3 and pip. Then run:

```
pip install Flask PyGithub
```

## 2. Run the web app

From the `repo-creator-web` directory, run:

```
python app.py
```

## 3. Access the web form

Open your browser and go to:

```
http://127.0.0.1:5000/
```

Fill out the form to create and configure a new GitHub repository.

---

You can later deploy this app to Google Cloud Run, App Engine, or another cloud service for team-wide access.
